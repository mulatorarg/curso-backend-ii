# Proyecto

## Comandos iniciales

Descargamos el proyecto.

```sh
git clone https://github.com/mulatorarg/curso-backend-ii.git
```

Ingresamos a la carpera recientemente descargada y luego instalamos los paquetes requeridos.

```sh
cd curso-backend-ii
npm i
```

Ejecutamos
```sh
npm run dev
```
